The code is implemented in MATLAB R2018b on Windows 10. To get started quickly, run 'main.m'.

When using your own data, two parts are required:

1) A ‘V × N’ matrix, where V is the number of images and N is the number of predictions obtained by applying a trained model to N different transformed versions of each image.

2) A ‘V × 1’ vector containing the ground truth value for each image.

Note: This code is for research purposes only. Refer to the following publication for details:

Yibiao Rong, Zehua Jiang, Zhemin Zhuang, Zhun Fan, Haoyu Chen. Uncertainty quantification for convolutional neural networks based on test  time augmentation with multiobjective evolutionary algorithms, Apploed Soft Computing, 202 (2026): 115799.
